Health and Usage Monitoring System

Sense Data

Implement our simple Data-Input API and begin sensing data today. Push data in a variety of formats using our built in translation tools or create your own bespoke translation plug-in.

Analyse Data

Define how you want your data to be analysed. Take advantage of built in analysis tools providing a standard set of analysis functions. For more complex analysis GPIG-C allows you to plug in your own analysis components.

Report Data

Pull data reports at your pleasure in a variety of formats including graphs, PDF and XML. Recieve event notifications when defined trends in your data have been identified. You can use the built in reporting and notification tools or build your own


GPIC-C HUMS Solution Enables You To

  • Push a wide variety of data types.
  • Take advantage of optional built in data persistance.
  • Use the GPIG-C Admin centre on your business intranet or online.
  • Define rules for data analysis using our built in rules engine.
  • Create your own bespoke analysis tools and plug them in to GPIG-C.
  • Define what constitutes an event within your data and what to do when events occur.
  • Send out notifications to alter the behaviour of your system when events occur.
  • Send external notifications to alert users of system issues.
  • Pull data analysis reports at any time, in a variety of formats including PDF, XML and Graphs.

For An Integrated Offline Solution

  • Use GPIG-C as your HUMS framework providing your device .
  • Still take advantage of built in plugins for data input, analysis tools, reporting and notification systems.
  • Plug in your own data persistance module.
  • Modify your system behaviour based on data events.
  • Perform realtime data analysis or batch.
